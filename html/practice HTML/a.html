Java Interview Questions & Answers

1. What is JDK, JRE, and JVM?

JDK (Java Development Kit): A full-featured software development kit used to develop Java applications. It includes the JRE plus development tools like the compiler and debugger.

JRE (Java Runtime Environment): The environment required to run Java applications. It includes JVM and standard Java class libraries.

JVM (Java Virtual Machine): A virtual machine that executes Java bytecode, providing platform independence.

2. What is Java?

Java is a high-level, object-oriented programming language designed for portability and platform independence. It's widely used for web, mobile, and enterprise applications.

3. What are data types in Java?

Java has two categories of data types:

  • Primitive data types: byte, short, int, long, float, double, boolean, char.
  • Reference/Object types: Instances of classes, arrays, and interfaces.

4. What is a variable?

A variable is a named storage location in memory that holds data. Java variables must be declared with a data type and can be initialized with a value. Types include local, instance, and class (static) variables.

5. What are looping statements?

Looping statements repeat a block of code while a condition is true:

  • for loop: Executes code a specific number of times.
  • while loop: Executes code while the condition remains true.
  • do-while loop: Executes code at least once, then repeats while true.
  • enhanced for loop: Used to iterate over arrays or collections.

6. What are conditional statements?

Conditional statements control program flow based on conditions:

  • if statement: Executes a block if a condition is true.
  • if-else statement: Executes code blocks based on true/false.
  • switch statement: Selects one of many blocks based on a value.

7. What is an array?

An array is a container object that holds a fixed number of values of the same type. Elements are accessed by index, starting at 0. Arrays can be single or multi-dimensional.

8. What is a String?

A String is an object in Java that represents a sequence of characters. It is immutable, meaning once created, it cannot be changed. Common methods include length(), substring(), and equals().

9. What are methods?

Methods are blocks of code that perform specific tasks, defined inside classes. They can take parameters, return values, and be overloaded or overridden.

10. What are operators?

Operators are symbols that perform operations on variables and values. Types include:

  • Arithmetic: +, -, *, /, %
  • Relational: ==, !=, >, <, >=, <=
  • Logical: &&, ||, !
  • Bitwise: &, |, ^, ~, <<, >>, >>>
  • Assignment: =, +=, -=, etc.

11. What are keywords?

Keywords are reserved words that have predefined meanings in Java syntax, such as class, public, static, void, if, else, new, and more.

12. What are classes and objects?

A class is a blueprint/template for creating objects. It defines data (fields) and behaviors (methods).

An object is an instance of a class containing actual values and state.

13. What is a constructor?

A constructor is a special method used to initialize new objects. It has the same name as the class and no return type.

14. What is inheritance?

Inheritance is a mechanism where one class acquires the properties and behaviors (methods) of another class, promoting code reuse.

15. What is polymorphism?

Polymorphism allows objects to be treated as instances of their parent class rather than their actual class. It enables methods to perform differently based on the object's actual subclass.

16. What is encapsulation?

Encapsulation is the practice of hiding internal object details and exposing only what's necessary through public methods, improving modularity and security.

17. What is exception handling?

Exception handling is a construct to manage runtime errors using try, catch, finally, and throw to gracefully handle error conditions and maintain program flow.
